Locar Reference

Una Local Reference es una referencia que se crea directamente en la plantilla HTML y permite acceder a un elemento del DOM (como un input, button, etc.) dentro de la misma plantilla.

Esto es útil para interactuar con elementos del DOM sin necesidad de manipularlos directamente desde el código TypeScript del Componente.
